# Aplicación de Estadística y Probabilidad

## Descripción General
Esta aplicación web permite a los usuarios ingresar datos numéricos, generar datos aleatorios y aplicar conceptos fundamentales de probabilidad y estadística. Proporciona varios cálculos estadísticos y representaciones gráficas para mejorar la comprensión y el análisis de los datos.

## Características
- **Ingreso Manual de Datos**: Los usuarios pueden ingresar un mínimo de 20 datos numéricos.
- **Generación de Datos Aleatorios**: Los usuarios pueden generar un mínimo de 20 datos aleatorios.
- **Cálculos Estadísticos**: La aplicación calcula:
  - Media
  - Mediana
  - Moda
  - Mínimo
  - Máximo
  - Rango
- **Tabla de Frecuencias**: Muestra frecuencias absolutas y relativas, así como frecuencias acumuladas.
- **Representaciones Gráficas**: Incluye:
  - Histograma
  - Polígono de Frecuencias
  - Ojiva
  - Diagrama de Pareto
- **Cálculos de Probabilidad**: Presenta operaciones con conjuntos, cálculos de probabilidad y funciones combinatorias.
